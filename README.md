## Descripción del proyecto

Este proyecto investiga la relación entre el tipo de bolsa utilizada para transportar carga y la asimetría muscular en estudiantes universitarios que utilizan transporte público en Lima. Utilizamos electromiografía de superficie (sEMG) para evaluar la actividad del músculo trapecio en ambos lados del cuerpo, comparando el uso de bolsas de hombro único versus mochilas de doble correa. Los datos se analizan mediante el Índice de Asimetría Muscular (IAM) y métricas de amplitud (RMS) para determinar cómo incluye el tipo de carga en la activación muscular del músculo trapecio. De esta manera, se busca indagar sobre las implicaciones para la salud musculoesquelética de la población estudiantil que frecuenta el transporte público.
